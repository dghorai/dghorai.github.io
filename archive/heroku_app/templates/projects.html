Handwritten Text Recognition using Tensorflow, Keras, and IAM Dataset

Handwritten text contained in scanned images converted into digital text using Deep Learning. The handwritten text recognition model architecture consisting of CNN, RNN, and CTC Loss function. CNN is used to extract features from the handwritten text image. RNN is used to predict sequential output per time step. CTC Loss function is used to predict output for each time-step.
